# Spec Hub

Spec Hub helps product teams manage specs, roadmap decisions, and implementation handoff.

## Product shape

- a browser-based workspace for specs and feature reviews
- comments and approval flow for product and engineering
- release planning views for upcoming iterations

## Repository scope

This repository contains the main web application, the API, and background jobs for notifications and exports.

The following systems are outside this repository:

- centralized identity provider
- file storage service for large attachments
- notification delivery service used by multiple company products

## Immediate business goal

The team wants a cleaner planning baseline before expanding collaboration features in v1.
