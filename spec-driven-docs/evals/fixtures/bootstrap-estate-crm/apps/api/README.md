# API Service

The API service exposes:

- listing import endpoints for partner connectors
- CRM pipeline data for the internal dashboard
- lead assignment and task automation endpoints

## Integrations

- partner MLS feeds
- partner XML and CSV exports
- shared identity service
- billing webhooks from an external broker billing platform

## Current concerns

- partner feed quality is inconsistent
- managers want a clean audit trail for lead reassignment
- pilot agencies care more about import speed than advanced analytics
