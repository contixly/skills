# DealDesk CRM

DealDesk CRM is a B2B SaaS product for small and mid-sized real-estate agencies.

## What the product does

- imports property listings from multiple MLS and partner feeds
- helps agents assign leads and track deal stages
- gives managers a pipeline view across listings, deals, and agent workload

## Who uses it

- agency owners
- sales managers
- agents
- backoffice listing coordinators

## What this repository contains

- the internal web application for agency staff
- the API used by the web app and partner ingestion jobs
- background workers for listing import and CRM automations

## What is outside this repository

- public marketing website
- broker billing service managed by another team
- identity and SSO service shared across products

## Near-term delivery pressure

- launch an MVP for three pilot agencies in six weeks
- prove that listing import and CRM pipeline management work without spreadsheet exports
- keep the initial scope narrow enough for one product squad
